Hotel Management System

This project is a functional Hotel Management System built using Python and SQLite3. It serves as a simple yet effective CRUD (Create, Read, Update, Delete) application for managing hotel operations. The system includes features such as guest check-in, where user information is stored in a database, viewing a guest list, updating room availability during check-out, and deleting guest information post-checkout. Designed with a user-friendly interface, this project highlights the power of Python and SQLite3 in creating practical, database-driven applications for real-world use.
